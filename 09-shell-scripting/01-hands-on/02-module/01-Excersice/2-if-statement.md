# Implementing if statement

```
#!/bin/bash
#Initializing two variables
a=10
b=10

#Check whether they are equal
if [ $a == $b ]
then
    echo "a is equal to b"
fi

#Check whether they are not equal
if [ $a != $b ]
then
    echo "a is not equal to b"
fi
```

<Mark> Output </Mark>
```
a is equal to b
```
